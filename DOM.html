O que é o DOM?

DOM (Document Object Model) é uma representação estrutural do documento HTML como uma árvore de objetos. Cada elemento HTML se torna um nó na árvore, permitindo que linguagens de programação como JavaScript manipulem o conteúdo, estrutura e estilo da página.

Características do DOM:

  • Estrutura hierárquica: Os elementos são organizados em uma árvore de pais e filhos
  • Nós: Cada elemento, atributo e texto é um nó na árvore
  • Manipulação dinâmica: Permite adicionar, remover e modificar elementos
  • Interface de programação: Fornece métodos para acessar e modificar elementos

Árvore DOM da Página Principal (index.html)

<!-- Declaração do tipo de documento -->
<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title> Jogo de Damas - Projeto Web </title>
<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.ico" type="image/x-icon">
</head>
<body>
<header>
<h1> Jogo de Damas </h1>
<nav>
<ul>
<li> <a href="#sobre"> Sobre o Projeto </a> </li>
<li> <a href="#objetivos"> Objetivos </a> </li>
<li> <a href="#disciplina"> Disciplina </a> </li>
<li> <a href="#links"> Links do Projeto </a> </li>
<li> <a href="DOM.html"> Árvore DOM </a> </li>
</ul>
</nav>
</header>
<main>
<aside>
<h2> Informações do Projeto </h2>
<section class="integrantes">
<h3> Integrantes </h3>
<ul> /* Lista de integrantes */ </ul>
</section>
/* Outras seções do aside: curso, descrição, recursos, repositório */
</aside>
<div class="content">
<section id="sobre">
<h2> Sobre o Projeto </h2>
<p> Este projeto consiste no desenvolvimento... </p>
</section>
/* Outras seções: objetivos, disciplina, links */
</div>
</main>
<footer>
<address> /* Informações de contato */ </address>
<div class="links-uteis"> /* Links úteis */ </div>
<p class="copyright"> © 2024 Projeto Jogo de Damas... </p>
</footer>
</body>
</html>

Análise da Estrutura

Elementos Principais:

  • <html>: Elemento raiz do documento
  • <head>: Contém metadados, título e links para recursos externos
  • <body>: Contém todo o conteúdo visível da página
  • <header>: Cabeçalho com título e navegação
  • <main>: Conteúdo principal da página
  • <aside>: Barra lateral com informações complementares
  • <footer>: Rodapé com informações de contato e links

Elementos Semânticos Utilizados:

  • <nav>: Navegação principal
  • <section>: Seções temáticas do conteúdo
  • <address>: Informações de contato
  • <article>: Conteúdo independente (implícito nas seções)

Atributos Importantes:

  • id: Identificadores únicos para navegação interna
  • class: Classes CSS para estilização
  • href: Links de navegação
  • lang: Idioma do documento
